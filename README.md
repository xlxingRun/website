## 快速启动
下载项目
```shell
git clone https://github.com/xlxingRun/website.git
```

下载主题子项目
```shell
git submodule add https://github.com/escalate/hugo-split-theme.git themes/split
```

开发环境
```shell
hugo server -D
```

生产环境
```shell
hugo server -P
```

项目编译
```shell
hugo
```
